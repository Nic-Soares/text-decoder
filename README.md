## Getting Started

### Prerequisites

* npm

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repository

   ```sh
   git clone https://github.com/Nic-Soares/text-decoder.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Start development server

   ```sh
   npm run dev
   ```
